<h2>Comparação entre versão monolítica e versão modularizada.</h2>
A versão monolítica é muito desorganizada, por isso dificulta o entendimento da lógica do programa. Enquanto que a versão modularizada é de fácil leitura e manutenção e ainda permite que os métodos do programa sejam reutilizados nele ou mesmo em outro. Além disso, como os parâmetros vão na mesma linha da chamada do método, elimina-se a necessidade de uso de várias linhas.
